# learn-gh-actions

🚀 Learning GitHub Actions – GH-200

This repository contains practical examples, notes, and workflows to learn GitHub Actions (GH-200).
It covers the fundamentals of CI/CD automation using GitHub Actions with real workflow examples.

📚 Overview

GitHub Actions allows developers to automate software workflows directly inside a GitHub repository.
It helps in building, testing, and deploying code automatically.

This project is created to practice and understand:

GitHub Actions fundamentals

Workflow creation

CI/CD pipelines

Automation for builds and testing

Event-driven workflows

🎯 Learning Objectives

After completing this learning project, you will understand:

What GitHub Actions are

How workflows work

Creating YAML workflow files

Running jobs and steps

Using actions from the marketplace

Building CI pipelines

Automating tests

Deploying applications
